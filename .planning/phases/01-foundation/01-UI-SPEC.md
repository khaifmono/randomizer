| Destructive | `--destructive` | oklch(0.577 0.245 27.325) — red | "Clear history" button (ghost/text style, destructive color on hover only) |
| Muted text | `--muted-foreground` | oklch(0.553 0.013 58) — warm gray | History entry row secondary text (#N index prefix), empty state body copy, placeholder panel overlay |
| Border | `--border` | oklch(0.923 0.003 49) — light stone | Card borders, separator between tool area and history panel on desktop |

**New tokens to add to `@theme inline` block in `apps/web/src/index.css`:**
```css
--color-wheel-accent: oklch(0.55 0.18 240);
--color-dice-accent: oklch(0.55 0.15 145);
--color-coin-accent: oklch(0.72 0.15 80);
```

Accent reserved for:
1. Active tab underline indicator — one accent color per tab (wheel = blue, dice = green, coin = amber)
2. Active tab trigger icon — tinted to matching accent when that tab is selected
3. Nothing else in Phase 1 — no buttons, no backgrounds, no borders use accent

Placeholder panels (Dice, Coin in Phase 1): render at `opacity-50` using existing `--foreground` and `--muted` tokens. Do not apply accent colors to disabled/placeholder panels.

## Layout Contract

### Desktop (md and above)
- Page: `min-h-screen bg-background` standalone route, no sidebar
- Page header: centered, `py-12`, contains title "Randomizer Toolkit" (heading role, 20px semibold) and tagline "Spin, roll, or flip — decide anything." (body role, 14px regular, muted foreground)
- Split layout: `flex flex-row gap-8`
  - Tool area: `flex-1 min-w-0` — contains the tab bar + tab content
  - History panel: `w-64 shrink-0` — always visible, Card-wrapped

### Mobile (below md)
- Tool area: `w-full`
- History panel: hidden by default (`hidden`), toggled to `block` via `useState`
- Toggle button: sits inside tool area, top-right aligned (`flex justify-end mb-2 md:hidden`), minimum height 44px, "Show History" / "Hide History" label with `History` Lucide icon

### Tab bar
- `TabsList variant="line"` — underline indicator style (source: shadcn Tabs radix-vega style)
- Three triggers: Wheel, Dice, Coin
- Each trigger: Lucide icon (16x16) + text label, `gap-1` (4px) between icon and text
- Wheel trigger: active state uses `--color-wheel-accent` underline
- Dice trigger: active state uses `--color-dice-accent` underline; rendered as disabled in Phase 1 (greyed, pointer-events-none)
- Coin trigger: active state uses `--color-coin-accent` underline; rendered as disabled in Phase 1

### Tab icons
- Wheel: `RotateCcw` (16x16) — verify availability in lucide-react@0.562.0; fallback `RefreshCw`
- Dice: `Dices` (16x16) — verify availability; fallback `Square`
- Coin: `Coins` (16x16) — verify availability; fallback `Circle`

## Accessibility Contract

- Tab bar: Radix UI `Tabs` handles all keyboard navigation (arrow keys, Home/End), focus management, and ARIA roles. Do not override.
- Focus ring: inherits `outline-ring/50` from global base styles (`apps/web/src/index.css` `@layer base`)
- Disabled Dice/Coin triggers: use the `disabled` prop on `TabsTrigger` — Radix handles `aria-disabled` correctly
- History list: render as `<ul>` with `<li>` items for screen reader semantics
- History empty state: render as `<p>` (not `<div>`) for semantic meaning
- Mobile toggle button: must have accessible label "Show History" / "Hide History" matching visible text — no icon-only buttons
- Color contrast: all text uses `--foreground` or `--muted-foreground` on `--background` — inherits shadcn's WCAG AA baseline
