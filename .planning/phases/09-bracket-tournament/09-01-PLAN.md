    ```typescript
    export type BracketEntry = { id: number; name: string; isBye: boolean };
    export type Matchup = {
      id: string;           // "r{round}-m{index}"
      topEntry: BracketEntry | null;
      bottomEntry: BracketEntry | null;
      winnerId: number | null;
      isBye: boolean;
    };
    export type BracketState =
      | { phase: "entry" }
      | { phase: "playing"; rounds: Matchup[][]; activeMatchupId: string | null; animating: boolean }
      | { phase: "complete"; winnerId: number };
    ```

    2. **Pure functions** (exported for testing):
    - `nextPowerOf2(n: number): number` — returns 2/4/8/16 using if-chain (per RESEARCH.md)
    - `generateBracket(names: string[]): Matchup[][]` — clamp to 16, pad with BYE entries, Fisher-Yates shuffle, pair into round 0, auto-resolve bye matchups (set winnerId), build empty subsequent rounds
    - `advanceWinner(rounds: Matchup[][], roundIndex: number, matchupIndex: number, winner: BracketEntry): Matchup[][]` — immutable update; matchup i feeds round+1 matchup floor(i/2), top if even, bottom if odd

    3. **Hook** `useBracket()`:
    - State: `entries: string[]`, `mode: "random" | "judge"`, `bracketState: BracketState`, `history: HistoryEntry[]`
    - Refs: `isAnimatingRef`, `pendingWinnerRef`, `roundsRef` (mirrors rounds state), `nextIdRef`
    - `addEntry(name)`: push to entries if phase=entry
    - `setEntries(names[])`: replace entries if phase=entry
    - `setMode(m)`: set mode if phase=entry
    - `startTournament()`: guard isAnimatingRef; generate bracket; propagate all bye winners immediately via advanceWinner loop; find first non-bye matchup as activeMatchupId; if random mode, pre-determine winner for active matchup into pendingWinnerRef; set phase=playing
    - `resolveMatchup(matchupId, winnerId)`: judge mode only; set winnerId on matchup, advance winner, find next active matchup (or complete)
    - `onAnimationEnd(matchupId)`: apply pendingWinnerRef winner, advance, find next active; if random mode, pre-determine next winner; if no more matchups, complete tournament
    - `resetBracket()`: set phase=entry, clear rounds
    - `ANIMATION_DURATION = 1200` exported constant
    - Internal helper `findNextActiveMatchup(rounds)`: returns first matchup where winnerId is null and both entries are non-null and isBye is false; returns null if none
    - On tournament complete: find winner entry by winnerId from entries array, create HistoryEntry with label "Winner: {name} ({rounds.length} rounds, {entrantCount} entrants)", prepend to history

    4. **Update types.ts**: Add `"bracket"` to the TabId union: `type TabId = "wheel" | "dice" | "coin" | "number" | "teams" | "bracket";`
