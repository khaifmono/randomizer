    - Test: onRollEnd sets sum to correct total, creates history entry with format "N+N+N=sum", and clears rolling after 200ms settle delay

    RED phase: Create use-dice.test.ts with all tests above using @testing-library/react renderHook + act. Use vi.useFakeTimers() for the 200ms settle delay testing (same pattern as use-wheel.test.ts). All tests should fail initially.

    GREEN phase: Create use-dice.ts implementing the useDice hook:
    - State: count (default 2), rolling, results (number[]), sum (number | null), history (HistoryEntry[])
    - Refs: isRollingRef (synchronous guard), pendingResultsRef (pre-determined values), nextIdRef (history IDs)
    - setCount: clamp to 1-6, no-op when rolling
    - startRoll: guard with isRollingRef, pre-determine results with Array.from({length: count}, () => Math.ceil(Math.random() * 6)), set pendingResultsRef, setResults, setRolling(true)

    - Export: named export { useDice }
    - Import HistoryEntry from "./types"
