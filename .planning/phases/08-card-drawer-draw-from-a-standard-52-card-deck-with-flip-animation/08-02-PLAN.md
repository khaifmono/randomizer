    **cards-tab.tsx** — Container component.

    Props: `{ onHistoryChange: (entries: HistoryEntry[]) => void }`

    Wiring:
    1. `const { deck, drawnCards, isDrawing, mode, handSize, history, remainingCount, drawCards, onDrawEnd, reshuffle, setMode, setHandSize } = useCards();`
    2. `const [revealedCount, setRevealedCount] = useState(0);`
    3. Sync history up: `useEffect(() => { onHistoryChange(history); }, [history, onHistoryChange]);`
    4. Staggered flip animation (per D-05): When `isDrawing` becomes true, reveal cards one at a time:
       ```typescript
       useEffect(() => {
         if (!isDrawing) return;
         setRevealedCount(0);
         // Reveal each card with STAGGER_DELAY ms gap (200ms per D-05)
         drawnCards.forEach((_, i) => {
           setTimeout(() => setRevealedCount(i + 1), i * STAGGER_DELAY);
         });
         // Call onDrawEnd after all cards have flipped + settle (200ms extra)
         const total = drawnCards.length * STAGGER_DELAY + ANIMATION_DURATION + 200;
         const timer = setTimeout(onDrawEnd, total);
         return () => clearTimeout(timer);
       }, [isDrawing]);
       // Note: exclude drawnCards, onDrawEnd from deps (same pattern as CoinTab's onFlipEnd exclusion)
       ```
       Where `STAGGER_DELAY = 200` (const in file).
    5. Reset revealedCount when new draw starts: handled by `setRevealedCount(0)` at top of isDrawing effect.

    Render:
    ```tsx
    <div className="flex flex-col items-center gap-6 w-full max-w-md">
      {drawnCards.length > 0 && (
        <CardDisplay drawnCards={drawnCards} isDrawing={isDrawing} revealedCount={revealedCount} />
      )}
      <CardControls
        mode={mode}
        handSize={handSize}
        isDrawing={isDrawing}
        remainingCount={remainingCount}
        onDraw={drawCards}
        onReshuffle={reshuffle}
        onSetMode={setMode}

    </div>
    ```
