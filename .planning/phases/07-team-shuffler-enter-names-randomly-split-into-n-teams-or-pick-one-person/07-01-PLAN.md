  <behavior>
    Hook state shape:
    - names: string[]                 — current name list (user-entered)
    - mode: "pick-one" | "split"      — current operation mode
    - teamCount: number               — number of teams (2-8, only relevant in split mode)
    - shuffling: boolean              — true during animation window
    - result: ShuffleResult | null    — null until first shuffle
    - history: HistoryEntry[]         — newest first

    ShuffleResult type (local to use-teams.ts):
    - mode: "pick-one" | "split"
    - picked?: string                 — set in pick-one mode
    - teams?: string[][]              — set in split mode, array of arrays

    Actions:
    - setNames(names: string[]): void        — replaces name list, no-op when shuffling
    - setMode(mode: "pick-one" | "split"): void — no-op when shuffling
    - setTeamCount(n: number): void          — clamps 2-8, no-op when shuffling
    - startShuffle(): void                   — guard -> pre-determine -> set shuffling=true
    - onShuffleEnd(): void                   — read pending -> compute result -> append history -> setTimeout 200ms clear

    Split algorithm (Fisher-Yates shuffle then chunk):
    1. Copy names array, Fisher-Yates shuffle in place
    2. Distribute into teamCount buckets round-robin (index % teamCount)
       so teams are as even as possible
    3. Return array of arrays (empty teams are allowed if names < teamCount)

    Pick-one algorithm:
    - Math.floor(Math.random() * names.length) to pick index

    History label formats:
    - pick-one: "Picked: {name}"
    - split:    "Teams of {n}: [{name, name}] [{name, name}] ..."
      where n = Math.ceil(names.length / teamCount) (max team size)
      and each bracket contains comma-separated names of that team

    Test cases:
    - Initial state: names=[], mode="split", teamCount=2, shuffling=false, result=null, history=[]
    - setNames(["Alice","Bob","Carol"]) updates names
    - setMode("pick-one") updates mode
    - setTeamCount(3) updates teamCount; setTeamCount(1) no-op; setTeamCount(9) no-op
    - startShuffle sets shuffling=true
    - startShuffle is a no-op when shuffling=true (double-call guard)
    - setNames no-op when shuffling=true
    - setMode no-op when shuffling=true
    - setTeamCount no-op when shuffling=true
    - onShuffleEnd (pick-one): result.mode="pick-one", result.picked is one of the names, history has 1 entry with "Picked:" label
    - onShuffleEnd (split, 4 names, 2 teams): result.teams has 2 arrays, all 4 names distributed, no duplicates
    - onShuffleEnd clears shuffling after 200ms (fake timers)
    - onShuffleEnd with empty names list: pick-one sets picked=undefined, split sets teams=[]
    - History prepend: second shuffle result is history[0], first is history[1]
    - History IDs increment
  </behavior>

       ANIMATION_DURATION = 800

       setNames: if (isShufflingRef.current) return; setNamesState(names);
       setMode: if (isShufflingRef.current) return; setModeState(mode);
       setTeamCount: if (isShufflingRef.current) return; if (n < 2 || n > 8) return; setTeamCountState(n);

       startShuffle:
         if (isShufflingRef.current) return;
         isShufflingRef.current = true;
         let pending: ShuffleResult;
         if current mode === "pick-one":
           if names is empty: pending = { mode: "pick-one", picked: undefined };
           else: pick = names[Math.floor(Math.random() * names.length)]; pending = { mode: "pick-one", picked: pick };
         else (split):
           if names is empty: pending = { mode: "split", teams: [] };
           else:
             shuffled = [...names]; Fisher-Yates on shuffled;
             teams: string[][] = Array.from({ length: teamCount }, () => []);
             shuffled.forEach((name, i) => teams[i % teamCount].push(name));
             pending = { mode: "split", teams };
         pendingResultRef.current = pending;
         setShuffling(true);

       onShuffleEnd:
         const pending = pendingResultRef.current!;
         setResult(pending);
         let label: string;
         if pending.mode === "pick-one":
           label = pending.picked ? "Picked: " + pending.picked : "Picked: (none)";
         else:
           const maxSize = pending.teams!.length > 0 ? Math.ceil(names.length / teamCount) : 0;
           const brackets = pending.teams!.map(t => "[" + t.join(", ") + "]").join(" ");
           label = "Teams of " + maxSize + ": " + brackets;
         const entry: HistoryEntry = { id: nextIdRef.current++, label, timestamp: Date.now() };
         setHistory(prev => [entry, ...prev]);
         setTimeout(() => { isShufflingRef.current = false; setShuffling(false); }, 200);
