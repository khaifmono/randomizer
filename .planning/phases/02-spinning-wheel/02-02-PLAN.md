    Create `apps/web/src/components/randomizer/wheel/wheel-canvas.tsx` with the following structure:

    **Props interface:**
    ```typescript
    type WheelCanvasProps = {
      items: string[];
      spinning: boolean;
      winnerIndex: number | null;
      winner: string | null;
      onSpin: () => void;
      onSpinEnd: () => void;
    };
    ```

    **Refs:**
    - `canvasRef = useRef<HTMLCanvasElement>(null)`
    - `rotationRef = useRef(0)` — current rotation angle in radians
    - `rafRef = useRef(0)` — requestAnimationFrame ID for cleanup
    - `ctxRef = useRef<CanvasRenderingContext2D | null>(null)` — cached context
    - `cssSizeRef = useRef(0)` — cached CSS pixel size

    **DPR-aware Canvas initialization (Pattern 3 from RESEARCH):**
    Use a `useEffect` that:
    1. Gets `canvas = canvasRef.current`
    2. Computes `dpr = window.devicePixelRatio || 1`
    3. Uses `ResizeObserver` on canvas parent to track size changes
    4. On resize callback: reads `cssSize = entry.contentRect.width`, sets `canvas.width = cssSize * dpr`, `canvas.height = cssSize * dpr`, gets context, calls `ctx.scale(dpr, dpr)`, stores in refs
    5. Returns cleanup that disconnects ResizeObserver and cancels rAF

    **Draw loop (runs continuously via requestAnimationFrame):**
    Create a `drawWheel` function that:
    1. Clears canvas: `ctx.clearRect(0, 0, cssSize, cssSize)`
    2. Computes center `cx = cy = cssSize / 2`, radius `= cssSize / 2 - 4`
    3. If `items.length === 0`: draw a single circle with fill `#e5e5e5` (muted), center text "Add items to spin" in `14px` font, muted-foreground color `#737373`, return early
    4. For each item at index `i`:
       - `startAngle = rotationRef.current + i * segmentAngle`
       - `endAngle = startAngle + segmentAngle`
       - Fill segment with `getSegmentColor(i)` from wheel-math.ts
       - Stroke segment border: white at 40% opacity (`rgba(255,255,255,0.4)`), lineWidth 1.5
       - Draw radial text: save context, translate to center, rotate to `midAngle = startAngle + segmentAngle/2`, set `ctx.textAlign = "right"`, fill white (`#ffffff`), shadow for legibility (`ctx.shadowColor = "rgba(0,0,0,0.5)"`, `ctx.shadowBlur = 2`), font size adaptive: `Math.max(11, Math.min(14, radius / items.length * 1.5))px`, truncate text to 16 chars with ellipsis, `ctx.fillText(text, radius - 10, 4)`, restore context
    5. Draw center hub: white circle at center, radius 14px
    6. Schedule next frame: `rafRef.current = requestAnimationFrame(draw)`

    Start the draw loop in a `useEffect` that depends on `[items]`. Return cleanup: `cancelAnimationFrame(rafRef.current)`.

    **Motion animation (useAnimate from motion/react — Pattern 2 from RESEARCH):**
    Use a `useEffect` watching `[spinning]`:
    - If `spinning === false` or `winnerIndex === null` or `items.length === 0`: return early
    - Compute `fullRotations = 4 + Math.floor(Math.random() * 3)` (4-6 rotations per D-06)
    - Call `calculateStopAngle(rotationRef.current, winnerIndex, items.length, fullRotations)` from wheel-math.ts
    - Compute `duration = 4 + Math.random()` (4-5 seconds per D-06)
    - Use `animate(rotationRef, targetAngle, { duration, ease: [0.12, 0, 0.39, 0], onComplete: () => onSpinEnd() })`

    IMPORTANT: The `animate()` from `motion/react`'s `useAnimate()` is designed for DOM elements, NOT refs. Instead, use the imperative `animate()` function imported directly from `motion`:
    ```typescript
    import { animate } from "motion";
    ```
    Then in the spin effect:
    ```typescript
    const controls = animate(rotationRef.current, targetAngle, {
      duration,
      ease: [0.12, 0, 0.39, 0],
      onUpdate: (latest) => { rotationRef.current = latest; },
      onComplete: () => { onSpinEnd(); },
    });
    // Store controls ref for cleanup
    ```
    This drives the rotation value that the rAF draw loop reads each frame.

    **Pointer triangle (HTML overlay per RESEARCH — NOT drawn on canvas):**
    Render a container div with `className="relative"`:
    - Canvas element inside: `className="w-full aspect-square max-w-[480px]"`, cursor changes based on spinning state (`cursor-pointer` when idle + items present, `cursor-default` when empty, `cursor-not-allowed` when spinning), onClick calls `onSpin` only when not spinning and items.length > 0
    - Pointer div: absolute positioned at top center, CSS triangle pointing down using border trick:
      ```
      className="absolute top-0 left-1/2 -translate-x-1/2 -translate-y-0.5 w-0 h-0 pointer-events-none z-10"
      style={{ borderLeft: "12px solid transparent", borderRight: "12px solid transparent", borderTop: "22px solid white" }}
      ```
      Plus a slightly larger dark triangle behind it for the 2px stroke effect:
      ```
      style={{ borderLeft: "14px solid transparent", borderRight: "14px solid transparent", borderTop: "25px solid #1a1a1a" }}
      ```
      Or use a single SVG triangle: `<svg>` with white fill and dark stroke.
    - Winner overlay div: conditionally rendered when `winner !== null`:
      ```
      className="absolute inset-0 flex items-center justify-center pointer-events-none z-20"
      ```
      Inner div:
      ```
      className="px-6 py-3 rounded-xl animate-in fade-in zoom-in-95 duration-200"
      style={{ backgroundColor: "oklch(0.55 0.18 240 / 0.85)" }}
      ```
      Contains "Winner!" label (14px/600 semibold, white/80% opacity) and winner name (20px/600 semibold, white).

    **Export:** `export function WheelCanvas({ items, spinning, winnerIndex, winner, onSpin, onSpinEnd }: WheelCanvasProps)`
