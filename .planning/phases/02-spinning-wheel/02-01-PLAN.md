    - reset() restores items to the state they were in at initial load (before any spins removed items)

    - items returned by hook include both `items` (snapshot for canvas) and `liveItems` (current state for item list)

    Follow the hook skeleton from RESEARCH.md Pattern useWheel Hook Skeleton with these specifics:

    Constants:
    - `STORAGE_KEY = "wheel-items"`
    - `DEFAULT_ITEMS = ["Option 1", "Option 2", "Option 3", "Option 4"]`

    State:
    - `items` via `useState<string[]>(() => readStorage(STORAGE_KEY, DEFAULT_ITEMS))`
    - `originalItemsRef = useRef<string[]>(items)` — snapshot for reset
    - `itemsSnapshotRef = useRef<string[]>(items)` — frozen during spin animation
    - `isSpinningRef = useRef(false)` — synchronous guard (Pattern 4 from RESEARCH)
    - `winnerIndexRef = useRef<number | null>(null)`
    - `nextIdRef = useRef(1)`
    - `spinning` useState for UI disabled state
    - `winner` useState<string | null> for overlay display
    - `history` useState<HistoryEntry[]> for result history

    Functions (all wrapped in useCallback):
    - `setItems(next)`: calls `setItemsState(next)` and `writeStorage(STORAGE_KEY, next)`
    - `addItem(text)`: trim, reject empty, enforce 40 char max via `.slice(0, 40)`, append to items, update originalItemsRef
    - `addBulk(text)`: split on `\n`, trim each, filter empty, enforce 40 char max each, append all to items, update originalItemsRef
    - `removeItem(index)`: filter by index, setItems, update originalItemsRef
    - `reset()`: `setItems([...originalItemsRef.current])`
    - `startSpin()`: guard with isSpinningRef, freeze snapshot, pick winnerIndex via Math.floor(Math.random() * items.length), setWinner(null), setSpinning(true)
    - `onSpinEnd()`: read winnerIndexRef and itemsSnapshotRef, setWinner(won), filter winner from items, prepend to history with new HistoryEntry, setTimeout 2200ms then: setWinner(null), isSpinningRef.current=false, setSpinning(false)

    ```typescript
      winnerIndex: winnerIndexRef.current,
      history,
      addItem,
      addBulk,
      removeItem,
      reset,
      startSpin,
      onSpinEnd,
      hasRemovedItems,            // boolean: items.length < originalItemsRef.current.length — controls Reset button visibility
    }
    ```
